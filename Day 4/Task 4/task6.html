Welcome to the Main Content Area

This is the Holy Grail Layout — a classic web design pattern. It has a full-width header on top, a footer at the bottom, and three columns in the middle: a left sidebar, the main content area, and a right sidebar.

The layout uses CSS Flexbox to place the three middle columns side by side. The two sidebars have fixed widths, while the main content stretches to fill the remaining space using.

This is a great starting point for any webpage. You can replace this text with your own articles, images, or any other content!

Tips

Use flex: 1 on the main area so it fills leftover space.


Give sidebars a fixed width so they don't shrink.


Add min-height to the middle div so it looks good even with little content.
